---

# Sampling Process Diagram

```text id="sp7mko"

       ↓

       ↓
Sampling Method
       ↓
Sample Size
       ↓
Select Sample
       ↓

```

---

# Sample Size Calculation

## Formula

[
n = \frac{N}{1 + N(e)^2}
]

Where:

* N = Population size
* e = Margin of error

---

## Given Data

| Parameter           | Value  |
| ------------------- | ------ |
| Population (N)      | 100000 |
| Margin of Error (e) | 0.05   |

---

## Calculation

[
n = \frac{100000}{1 + 100000(0.05)^2}
]

[
0.05^2 = 0.0025
]

[
100000 \times 0.0025 = 250
]

[
1 + 250 = 251
]

[
n = \frac{100000}{251}
]

[
n \approx 398
]

---

# Final Answer

Required sample size = **398**

Probability sampling is a sampling method where every member has equal chance of selection.

* Random selection method.

### Example

* Selecting students using lottery method.

### Advantages

* Easy to implement
* Quick

### Disadvantages

* Pattern bias possible

* Population divided into groups.

### Example

* Students selected from departments.

### Advantages

* Better representation
* Accurate results

### Disadvantages

* Complex process

---

* Useful for large populations

4. Suitable for statistical analysis

# Disadvantages of Probability Sampling

3. Requires population list

### (a) Discuss Chi-square test.

From the data given below about treatment of 250 patients suffering from a disease, state whether the new treatment is superior to the conventional treatment. [10]

| Treatment    | Favourable | Non-Favourable | Total |
| ------------ | ---------- | -------------- | ----- |
| New          | 140        | 30             | 170   |
| Conventional | 60         | 20             | 80    |
| Total        | 200        | 50             | 250   |

(Consider χ² value at 0.05 significance level and degree of freedom 1 = 3.84)

#### Answer

# Chi-Square Test

Chi-Square test is a statistical method used to determine whether there is significant relationship between categorical variables.

---

# Formula

[
\chi^2 = \sum \frac{(O-E)^2}{E}
]

Where:

* O = Observed Frequency
* E = Expected Frequency

---

# Step 1: Hypothesis Formulation

## Null Hypothesis (H₀)

There is no significant difference between new and conventional treatment.

## Alternate Hypothesis (H₁)

New treatment is superior to conventional treatment.

---

# Step 2: Expected Frequencies

## Formula

[
E = \frac{(Row\ Total)(Column\ Total)}{Grand\ Total}
]

---

# Expected Frequency Table

| Treatment    | Favourable | Non-Favourable |
| ------------ | ---------- | -------------- |
| New          | 136        | 34             |
| Conventional | 64         | 16             |

---

# Step 3: Chi-Square Calculation

| Cell                        | O   | E   | (O-E)²/E |
| --------------------------- | --- | --- | -------- |
| New Favourable              | 140 | 136 | 0.12     |
| New Non-Favourable          | 30  | 34  | 0.47     |
| Conventional Favourable     | 60  | 64  | 0.25     |
| Conventional Non-Favourable | 20  | 16  | 1.00     |

---

# Total χ² Value

[
\chi^2 = 1.84
]

---

# Step 4: Decision

| Parameter     | Value |
| ------------- | ----- |
| Calculated χ² | 1.84  |
| Table χ²      | 3.84  |

Since:

[
1.84 < 3.84
]

* Accept Null Hypothesis (H₀)

---

# Conclusion

There is no significant evidence to prove that new treatment is superior to conventional treatment.

### (b) Two types of drugs were used on 6 and 5 patients for reducing weight. Apply two sample t-test. [10]

Drug A: 8, 10, 12, 9, 14, 13
Drug B: 7, 9, 14, 12, 8

(Consider t-value at 0.05 significance = 2.262)

#### Answer

# Two Sample t-Test

Two sample t-test is used to compare means of two independent samples.

---

# Step 1: Hypothesis

## Null Hypothesis (H₀)

There is no significant difference in effectiveness of Drug A and Drug B.

## Alternate Hypothesis (H₁)

There is significant difference in effectiveness of Drug A and Drug B.

---

# Step 2: Mean Calculation

## Drug A Mean

[
\bar{X_1} = \frac{8+10+12+9+14+13}{6}
]

[
\bar{X_1} = 11
]

---

## Drug B Mean

[
\bar{X_2} = \frac{7+9+14+12+8}{5}
]

[
\bar{X_2} = 10
]

---

# Step 3: Variance Values

| Drug   | Variance |
| ------ | -------- |
| Drug A | 5.6      |
| Drug B | 8.5      |

---

# Step 4: Formula

[
t = \frac{\bar{X_1}-\bar{X_2}}{\sqrt{\frac{s_1^2}{n_1}+\frac{s_2^2}{n_2}}}
]

---

# Step 5: Calculation

[
t = \frac{11-10}{\sqrt{\frac{5.6}{6}+\frac{8.5}{5}}}
]

[
= \frac{1}{\sqrt{0.93+1.7}}
]

[
= \frac{1}{\sqrt{2.63}}
]

[
= \frac{1}{1.62}
]

[
t = 0.62
]

---

# Step 6: Decision

| Parameter          | Value |
| ------------------ | ----- |
| Calculated t-value | 0.62  |
| Table t-value      | 2.262 |

Since:

[
0.62 < 2.262
]

* Accept Null Hypothesis (H₀)

---

# Conclusion

There is no significant difference in effectiveness between Drug A and Drug B.
