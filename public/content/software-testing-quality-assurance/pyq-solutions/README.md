# STQA PYQ Analysis & Strategy

> [!TIP]
> STQA is a scoring subject if you focus on the **Standard Diagrams** and **Comparison Tables**.

## High-Frequency Topics
1. **Verification vs Validation**: Asked in almost every alternate year.
2. **Selenium Suite**: WebDriver architecture is a 10-marker favorite.
3. **Black Box vs White Box**: Often asked as a detailed comparison.
4. **STLC Phases**: Essential for theory questions.
5. **Testing Levels**: Unit, Integration, System, and Acceptance.

## Strategy for Exams
- **Draw Diagrams**: Always draw the Selenium WebDriver architecture and STLC cycle.
- **Tables**: Use tables for comparisons (BBT vs WBT, Verification vs Validation).
- **Code Snippets**: For Selenium, mention basic commands like `driver.get()`, `findElement()`, etc.
