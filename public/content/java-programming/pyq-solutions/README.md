# Java PYQ Analysis & Strategy

> [!TIP]
> Java Practical and Theory both depend on **Syntax Accuracy**. Always close your brackets!

## High-Frequency Topics
1. **Multithreading**: Lifecycle of a thread and synchronization.
2. **Exception Handling**: Try-catch-finally and custom exceptions.
3. **Collections Framework**: ArrayList vs Vector, HashMap vs Hashtable.
4. **JDBC**: The 5 steps to connect to a database.
5. **Servlets & JSP**: Lifecycle methods and session management.

## Strategy for Exams
- **Code Snippets**: Write small, compilable snippets for concepts like `implements Runnable` or `extends Exception`.
- **Hierarchies**: Draw the Exception and Collection hierarchy diagrams.
- **Short Notes**: Focus on standard definitions for OOPs concepts (Encapsulation, Polymorphism).
