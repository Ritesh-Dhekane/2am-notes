# RM One-Night Revision (Study OS)

> [!TIP]
> Research Methodology is about the **Process** and **Logic**. Memorize the Flowcharts!

## 1. Research Types
- **Descriptive**: "What is happening?" (Surveys, fact-finding).
- **Analytical**: "Why/How?" (Using facts to analyze).
- **Applied**: Solving a specific practical problem.
- **Fundamental**: Theoretical, building knowledge base.
- **Quantitative**: Numbers and Statistics.
- **Qualitative**: Meanings, definitions, concepts.

## 2. Research Process
1. Define Problem
2. Literature Review
3. Formulate Hypothesis
4. Research Design
5. Data Collection
6. Data Analysis
7. Report Writing

## 3. Sampling
- **Probability**: Random selection (Simple, Systematic, Stratified, Cluster).
- **Non-Probability**: Non-random (Convenience, Judgmental, Quota, Snowball).

## 4. Data Collection
- **Primary**: Original (Interview, Questionnaire, Observation).
- **Secondary**: Already collected (Books, Journals, Reports).

## 5. Scaling Techniques
- **Nominal**: Just names/labels (e.g., Gender).
- **Ordinal**: Rank order (e.g., 1st, 2nd, 3rd).
- **Interval**: Equal distances (e.g., Temperature).
- **Ratio**: Absolute zero exists (e.g., Weight, Height).

## 6. Report Structure
- Preliminary Pages (Title, Abstract)
- Main Text (Intro, Methodology, Results, Discussion)
- End Matter (Bibliography, Appendices)
